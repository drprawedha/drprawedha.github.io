Preambule

Sehubungan dengan rencana perubahan operator hotel dari Swiss-Bel International menjadi Accor, diperlukan proses transformasi yang mencakup aspek branding, sistem operasional, integrasi teknologi, serta peningkatan infrastruktur pendukung. Seluruh pekerjaan dilakukan secara terstruktur untuk memastikan transisi berjalan lancar tanpa mengganggu operasional hotel dan pengalaman tamu.

1. Brand Design Migration

Migrasi desain brand mencakup penggantian seluruh identitas visual hotel sesuai standar Accor, baik untuk kebutuhan internal maupun eksternal.

Scope of Work

  • External Signage
  • Internal Signage
  • Room Amenities Design
  • Stationery & Collateral
  • Front Office Branding Material
  • F&B Branding Material
  • Digital Signage

Before Implementation

  • Obtain official Accor Brand Guidelines.
  • Conduct branding asset audit.
  • Identify replacement and modification requirements.
  • Select production and installation vendors.

During Implementation

  • Monitor production quality.
  • Verify compliance with Accor standards.
  • Minimize guest disruption during installation.

After Implementation

  • Perform quality inspection.
  • Conduct brand compliance audit.
  • Close outstanding defects.

2. PMS & POS System Migration

Migrasi sistem dilakukan dari PowerPro ke Opera PMS untuk Front Office serta Rhapsody untuk POS dan Back Office. Fokus utama adalah menjaga integritas data dan memastikan operasional hotel tetap berjalan selama proses transisi.

Opera PMS

  • Guest Profile Migration
  • Reservation Migration
  • Rate Configuration
  • Room Inventory Setup
  • User Access Management

Rhapsody

  • Outlet Setup
  • Menu Configuration
  • Inventory Configuration
  • Financial Mapping
  • User Role Configuration

Before Implementation

  • Audit existing PowerPro database.
  • Data cleansing and validation.
  • Prepare backup and rollback procedures.
  • Determine migration cut-off date.

During Implementation

  • Execute migration process.
  • Perform data validation.
  • Conduct User Acceptance Test (UAT).
  • Verify transaction processing.

After Implementation

  • Validate migrated data.
  • Monitor system stability.
  • Resolve post go-live issues.
  • Finalize migration report.

3. PMS Interface Installation

Interface antar sistem dipasang untuk memastikan seluruh platform dapat berkomunikasi secara real-time dan akurat sesuai standar operasional Accor.

Potential Integration Systems

  • Door Lock System
  • PBX Telephone System
  • IPTV
  • Payment Gateway
  • Channel Manager
  • Booking Engine
  • Accounting System
  • Guest Internet System

Before Implementation

  • Review all required integrations.
  • Prepare network architecture.
  • Coordinate with all vendors.

During Implementation

  • Install interfaces.
  • Perform connectivity testing.
  • Conduct end-to-end transaction testing.

After Implementation

  • Monitor integration stability.
  • Review interface logs.
  • Document final configurations.

4. Infrastructure Enhancement

Infrastruktur pendukung dipersiapkan melalui pekerjaan leveling, penataan ruang server, dan pemasangan sistem pemadam kebakaran FM-200 untuk melindungi perangkat kritikal hotel.

Scope of Work

  • Server Room Preparation
  • Rack Organization
  • Cable Management
  • Power Distribution Optimization
  • FM-200 Installation
  • Commissioning and Testing

Before Implementation

  • Conduct infrastructure assessment.
  • Prepare engineering design.
  • Verify compliance requirements.

During Implementation

  • Install FM-200 system.
  • Perform room leveling and organization.
  • Conduct commissioning tests.

After Implementation

  • Execute acceptance testing.
  • Issue certification documents.
  • Schedule preventive maintenance.

Project Success Criteria

  • Brand identity fully complies with Accor standards.
  • Opera PMS and Rhapsody operate successfully.
  • All interfaces function normally.
  • Infrastructure meets operational and safety requirements.
  • No significant disruption to hotel operations during transition.
  • All departments complete UAT and Go-Live approval.
