Preambule

Karena ada rencana dari owner untuk mengubah operator hotel, dari swiss-bel international ke accor. Maka jadilah project rebranding.

Task

  • Migrasi Desain Brand

    • Desain yang dirubah terkait internal dan eksternal sign, dan desain untuk room ammenities

  • Migrasi sistem Opera dan Rhapsody

    • Migrasi sistem dari Powerpro ke Opera PMS (untuk front office) dan Rhapsdy (Untuk POS dan Back Office) dilakukan dengan pendekatan presisi, memastikan setiap data, transaksi, dan alur kerja berpindah tanpa mengganggu ritme operasional hotel.

  • Installing Interface untuk Tiap PMS (Property Management System)

    • Menginisiasi interface antar Property Management System dipasang dan diuji secara sistematis agar seluruh sistem dapat saling berkomunikasi dengan akurat dan efisien

  • Install Leveling dan FM 200

    • Mempersiapkan infrastruktur dan perlindungan operasional, memastikan seluruh lingkungan teknologi bekerja stabil, aman, dan siap menghadapi kemungkinan risiko di masa mendatang
