Preamble

As part of the planned transition of hotel management from Swiss-Bel International to Accor, a comprehensive transformation project is required covering branding, operational systems, technology integration, and supporting infrastructure. The objective is to ensure a seamless transition while maintaining operational continuity and guest satisfaction throughout the implementation period.

1. Brand Design Migration

The brand migration project involves replacing all visual identities associated with the previous operator and aligning them with Accor brand standards. This includes both guest-facing and internal assets.

Scope of Work

  • External Signage
  • Internal Signage
  • Room Amenities Design
  • Stationery & Marketing Collateral
  • Front Office Branding Materials
  • Food & Beverage Branding
  • Digital Signage

Before Implementation

  • Obtain official Accor Brand Guidelines.
  • Conduct a complete branding asset audit.
  • Identify replacement, modification, and removal requirements.
  • Select approved production vendors.

During Implementation

  • Monitor production quality and installation progress.
  • Ensure compliance with Accor standards.
  • Minimize disruption to hotel operations and guests.

After Implementation

  • Perform quality inspection.
  • Conduct brand compliance verification.
  • Resolve outstanding defects.
  • Document completed installations.

2. PMS & POS System Migration

The existing PowerPro platform will be replaced with Opera PMS for Front Office operations and Rhapsody for POS and Back Office management. The migration process must ensure data integrity, operational continuity, and minimal service interruption.

Opera PMS

  • Guest Profile Migration
  • Reservation Migration
  • Rate Configuration
  • Room Inventory Setup
  • User Access Management

Rhapsody

  • Outlet Configuration
  • Menu Configuration
  • Inventory Setup
  • Financial Mapping
  • User Role Management

Before Implementation

  • Audit and validate existing PowerPro data.
  • Perform data cleansing activities.
  • Create full backup and rollback procedures.
  • Define migration cutover schedule.

During Implementation

  • Execute migration process.
  • Validate migrated data.
  • Conduct User Acceptance Testing (UAT).
  • Verify operational transactions.

After Implementation

  • Monitor system performance.
  • Resolve post Go-Live issues.
  • Validate financial and operational reports.
  • Complete migration documentation.

3. PMS Interface Installation

Interfaces between Opera PMS, Rhapsody, and supporting hotel systems must be installed and tested to ensure seamless communication across all operational platforms.

Integrated Systems

  • Door Lock System
  • PBX Telephone System
  • IPTV
  • Payment Gateway
  • Channel Manager
  • Booking Engine
  • Accounting System
  • Guest Internet System

Before Implementation

  • Review integration requirements.
  • Prepare network architecture.
  • Coordinate with all vendors.

During Implementation

  • Install and configure interfaces.
  • Perform connectivity testing.
  • Conduct end-to-end transaction testing.

After Implementation

  • Monitor interface stability.
  • Review system logs.
  • Finalize integration documentation.

4. Infrastructure Enhancement

Infrastructure improvements include server room preparation, equipment organization, and installation of the FM-200 fire suppression system to protect critical IT assets.

Scope of Work

  • Server Room Preparation
  • Rack Organization
  • Cable Management
  • Power Distribution Optimization
  • FM-200 Installation
  • Testing and Commissioning

Before Implementation

  • Conduct infrastructure assessment.
  • Prepare engineering design documentation.
  • Verify safety and compliance requirements.

During Implementation

  • Install FM-200 system.
  • Perform server room leveling and organization.
  • Execute commissioning tests.

After Implementation

  • Conduct acceptance testing.
  • Issue certification documents.
  • Establish preventive maintenance schedules.

Project Success Criteria

  • All branding complies with Accor standards.
  • Opera PMS and Rhapsody operate successfully.
  • All interfaces function as designed.
  • Infrastructure meets operational and safety requirements.
  • No significant service disruption occurs during transition.
  • All departments complete UAT and Go-Live approval.
