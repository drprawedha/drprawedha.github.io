Preambule

For years, guests at ANTARA Restaurant scanned a QR code placed on each table and opened a shared PDF document containing the entire menu. While practical and easy to maintain, the experience was far from ideal on smartphones. Guests often needed to zoom, scroll through multiple pages and manually search for dishes and beverages.

To improve accessibility and create a more premium digital experience, the menu was redesigned into a lightweight web application built using HTML, CSS, JavaScript and JSON. The objective was to provide a faster, cleaner and more intuitive experience without sacrificing maintainability.

Although functional, the PDF workflow lacked interaction and required additional effort from guests to navigate through menu categories.

Project Goals

  • Replace the PDF menu with a responsive website.
  • Improve user experience on smartphones.
  • Create category-based navigation.
  • Implement instant search functionality.
  • Provide a more premium visual presentation.
  • Simplify menu maintenance using JSON.
  • Prepare the foundation for future digital dining features.

Lessons Learned

  • PDF documents are practical but limited for mobile interaction.
  • Simple technologies can provide excellent user experiences.
  • JSON-based content simplifies maintenance.
  • Micro interactions significantly improve usability.
  • Mobile-first design should always be prioritized.
  • Small UX improvements can dramatically enhance perceived quality.
